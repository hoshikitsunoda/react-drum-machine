# react-sequencer
A web application to for musicians who want to easily create musical sequences.
# Usage
```
npm run bundle:watch
then, open index.html
```
# Use
* HTML5
* CSS3
* React
* React-DOM
* Babel
* Babelify
* Watchify
* Browserify
* Stable-Timer
* Modern JavaScript(ES2015+)
* Node.js
